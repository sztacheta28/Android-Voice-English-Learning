# Nauka angielskiego ze słuchu
## 1. Opis
Quizz z 1000 angielskich słówek. Aplikacja posiada bazę popularnych angielskich słów. Co każde wylosowanie słowa jego wymowa pobierana jest z zewnętrznego serwera. Użytkownik na podstawie wymowy ma odgadnąć polskie znaczenie słowa. Po wybraniu odpowiedzi pojawia się także angielski zapis słowa.
